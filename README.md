
# 모두의 입시 레일

## 1️⃣ 엑셀 → JSON 변환 규칙
엑셀 열을 아래 컬럼으로 정리 후 JSON으로 변환하세요.

|컬럼|설명|
|---|---|
|college|대학교명|
|track|전형명|
|unit|모집단위|
|year|연도|
|metric|지표명(70%컷, 등록자내신평균 등)|
|value|값(숫자)|

JSON 예시:
{
 "college":"서강대학교",
 "track":"지역균형",
 "unit":"컴퓨터공학과",
 "year":2026,
 "metric":"70%컷",
 "value":1.96
}
